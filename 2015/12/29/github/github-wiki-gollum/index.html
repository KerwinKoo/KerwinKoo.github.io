开源wiki系统:gollum

由于此文章本身是在wiki中编写，因此不能写入此宏代码，需要将左边第一个"{"符号改为"["。

重启gollum即可

增加主页全文件目录

新建Home.md后，在此文件中写入以下：

    {<GlobalTOC()>> #将左边第一个{改为<

同理，需要将左边第一个{改为`<`

#### 创建文件目录

在点击`new`创建新文件后，在文件名中增加`/`来创建新的目录。如创建目录`tec`下的文件`test.md`，则在文件名中输入：

    tec/test

之后会自动创建tec下的文件test.md
