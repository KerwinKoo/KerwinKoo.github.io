Writing The Disk Image Into Raspberry Pi 2

  • arm
  • linux
  • openelec
  • raspberry
linux

OpenELEC 系统实现了一个很小的Linux内核中(包含少许必要的包,python2.7 SSH等...)内嵌 CODI 视频播放终端。本日志记录在Arm开发板，Raspberry Pi 2 中实现OpenELEC的基础镜像安装。

0.硬件准备:

  • 硬件平台:Raspberry Pi 2
  • 显示设备:支持HDMI显示器.(如果不支持HDMI,需要使用HDMI TO VGA转接头)
  • 存储设备:C10 8G TF card(TF Card 读取速度直接影响播放流畅性). TF card读卡器.

1.镜像下载:

openelec官网 下载镜像,要注意选择是树莓派2 B版 镜像下载.

由于目前没有对Openelec 或 CODI做任何定制化开发,因此还没有开始使用 OpenElEC的源码做交叉编译.关于定制化开发,日后研究.

2.镜像写入.

进入镜像下载后存放的目录,解压,或通过命令解压:

gunzip -d OpenELEC-RPi2.arm-5.0.8.img
